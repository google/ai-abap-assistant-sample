# About this repository

This repository contains code samples, and other artifacts related to the below product: 

| Product Name              | Description   | Sub-directory | Setup Guide |
| ------------------------- |---------------| ------------- | ------------- |
| AI ABAP Assistant | The AI ABAP Assistant is an open-source tool that integrates Google Cloud's Vertex AI (Gemini) directly into SAP GUI and Eclipse ADT, empowering developers to write, understand, and improve ABAP code faster. It acts as an intelligent coding partner that automates routine tasks such as generating boilerplate code and unit tests, explaining complex legacy logic, and performing automated code reviews directly within your development environment.  | [code](src) | [AI ABAP Assistant Setup Guide](transport_files) |

## Pre-requisite

 - Your SAP system must have Google ABAP SDK version 1.9 or higher installed. If not, please follow the installation guide https://cloud.google.com/solutions/sap/docs/abap-sdk/on-premises-or-any-cloud/latest/install-config
 - Please review the enhancements in the [code](src) before importing into your system.
